What is EXIF Metadata?

EXIF stands for Exchangeable Image File Format. a standard for storing supplemental metadata in digital photos. Each time you take a picture, your phone or camera saves a file (typically a JPEG) to your device's storage. This file contains all the data dedicated to the actual picture (pixels which you can see). Such metadata can hold date, time, camera settings, copyright information.

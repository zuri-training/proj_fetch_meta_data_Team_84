What is metadata?

Metadata is information about data, it enriches the data with information that makes it easier to find, use and manage.

Do i have to register before i can upload my files?

Yes, you have to register in order to gain full access. Creating an account allows you to upload your files, extract your metadata, save and download the extracted metadata.

What type of files can i upload?

You can only upload Images (JPG, PNG), PDF, CSV, JSON files for now.

Can i save my file metadata?

Yes, you can. 84meta allows you to save the extracted data into our database.

What other features are available?

You can upload up to three files at once. Cool right! This 84meta feature helps you to save a lot of time.
